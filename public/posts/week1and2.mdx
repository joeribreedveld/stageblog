Collezione is een website waar je nieuwe games kan vinden en games kan opslaan. Ook kan je games van andere mensen bekijken en game suggesties krijgen.

## Description

- Voor state management en global state heb ik gebruik gemaakt van Redux Toolkit met Collection slices en User slices voor alle actions. Ik heb de global state gebruikt voor de inhoud van de collectie van de gebruiker en om bij te houden of iemand is ingelogd.
- Voor collectie aanpassingen heb ik gebruik gemaakt van input fields die altijd in beeld staan waar je gewoon op kan klikken om dingen aan te passen. Op deze manier heb je geen extra knoppen of pagina's nodig en het ziet er redelijk clean uit.
- Voor een paar grote aanpassingen heb ik gebruik gemaakt van aparte branches. Ik heb gemerkt dat dit niet altijd nodig was. De meeste dingen zijn naar main gepusht. Als je in je eentje werkt aan 1 feature per keer is het niet nodig naar mijn mening.

### Tech Stack

- Next.js - Project structure and easy setup
- SASS (SCSS) - Advanced CSS styling techniques
- React.js - Reusable UI components
- Redux (Toolkit) - Global state management
- TypeScript - Type safe components in React.js

## Getting Started

### Dependencies

- Node.js (latest)

### Installing

- Clone GitHub project

```
## HTTPS
https://github.com/joeribreedveld/collezione.git

## SSH
git@github.com:joeribreedveld/collezione.git

## GitHub CLI
gh repo clone joeribreedveld/collezione
```

- Run install command

```bash
npm install
# or
yarn install
```

### Executing program

```
npm run dev
## or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) in je browser om resultaat te zien.
